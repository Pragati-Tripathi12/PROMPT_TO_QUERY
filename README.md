# 🧠Prompt to SQL Query Generator

Convert natural language into SQL queries using AI 

## Overview
This project allows users to input plain English prompts and automatically generates corresponding SQL queries. It simplifies database interaction for non-technical users and speeds up query writing for developers.

## ✨ Features
- 🔤 Convert natural language → SQL queries
- 🗄️ SQLite database integration
- ⚡ Fast and simple UI using Streamlit
- 🔐 Secure API key handling using `.env`

## 🛠️ Tech Stack
- Python 🐍
- Streamlit
- SQLite
- dotenv
- Google Generative AI (Gemini API)
